Note About SVG Icons

This example uses emoji placeholders. When you have your SVG icons ready:

  • Replace emoji with actual SVG files
  • Use the same class structure
  • SVG icons will automatically inherit colors and sizes
  • Place SVG files in the components/atoms/icons/ folder
