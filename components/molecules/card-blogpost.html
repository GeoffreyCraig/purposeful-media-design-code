Blog Card Component Documentation

  • Component Type: Molecule
  • Purpose: Display blog post preview with title, metadata, excerpt, and read more link
  • States: Default, Hover, Active
  • Fully Tokenized: All styles use design system variables from variables-v4.0-20251027.css
  • Responsive: Adapts typography and spacing across all breakpoints (Mobile, Tablet, Desktop, DesktopPlus)
  • Interactive: Hover changes background color, link color, and applies shadow
  • Clickable: Entire card is clickable for better UX
  • Accessibility: Uses semantic HTML with article and heading tags

Responsive Behavior

  • Mobile (380-767px): Title 16px, compact padding (12px 16px)
  • Tablet (768-1149px): Title 18px, medium padding (12px 16px)
  • Desktop (1150-1919px): Title 21px, standard padding (12px 16px)
  • DesktopPlus (1920px+): Title 21px, standard padding (12px 16px)

Usage

  • Used within blog-group organism to display multiple posts
  • Can be used standalone or in groups
  • Integrate with WordPress dynamic content via PHP
  • Card background changes on hover to accent-50 (light gold tint)
  • Text colors transition from navy to gray on hover for better readability
