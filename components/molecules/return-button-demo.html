Implementation Details

Button Specifications

  • Button Size: 48px × 48px
  • Icon: ui-icon-display molecule (large = 32px)
  • Position: Fixed to bottom-right (2rem from edges)
  • Background: White with 90% opacity
  • Border Radius: 8px
  • Shadow: Subtle elevation effect with hover enhancement

Icon Configuration

  • Component: ui-icon-display molecule
  • Icon: arrow-up
  • Size: large (32px)
  • Color: gray (default), navy (hover)
  • Direction: up

Code Example

<button class="return-button">
    <div class="ui-icon-display ui-icon-display--large" 
         data-icon="arrow-up" 
         data-color="gray" 
         data-direction="up"></div>
</button>
