Usage Notes

Component Structure

  • Fixed Size: 350px × 453px (portrait letter aspect ratio)
  • Gold Border: 4px solid border using accent-500 color
  • Overlay: Semi-transparent navy overlay (25% opacity) for text readability
  • Three Content Sections: Header (icon + headline), CTA paragraph, and button

Icon System

  • Uses the icon-display molecule component
  • Icons are displayed at 90×90px (large size variant)
  • Always use data-color="white" for visibility on dark overlay
  • Icon selection via data-icon attribute
  • Controlled by molecules-icon-display.css stylesheet

Typography

  • Headline: Lato Heavy, 21px, uppercase, white
  • CTA Text: Open Sans Regular, 18px, centered, white
  • Button: Lato Bold, 18px, uppercase, navy on gold background

Button States

  • Default: Gold background (#D4AF37)
  • Hover: Light gold (#DCBE5D)
  • Active: Dark gold (#B39226)
  • Disabled: Light gold with 50% opacity
  • Focus: Gold outline for accessibility

CSS Dependencies

  • Requires variables-v2.0-20250915.css for color and typography tokens
  • References atoms-master.css for base atomic styles
  • Requires molecules-icon-display.css for icon display molecule
  • Icon sprite sheet handled automatically by icon-display component

Accessibility Considerations

  • Always include meaningful alt text for thumbnail images
  • Button includes focus state for keyboard navigation
  • Color contrast meets WCAG AA standards with overlay
  • Semantic HTML structure with proper heading hierarchy
