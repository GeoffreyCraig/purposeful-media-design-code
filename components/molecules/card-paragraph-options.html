Usage Notes

Component Structure

  • 12 variants total: 4 breakpoints (DesktopPlus, Desktop, Tablet, Mobile) × 3 colors (Default, Gray, Reverse)
  • Data attributes:
    • data-breakpoint: Controls layout and sizing
    • data-color: Controls color scheme
    • data-show-icon: Boolean toggle for icon visibility ("true" or "false")
  • Icon system: Uses icon-display molecule with data-attributes

Layout Behavior

  • DesktopPlus/Desktop/Tablet: Horizontal layout with icon on left
  • Mobile: Vertical layout with centered icon and content

Key Implementation Notes

  • HTML Structure: The component now uses a two-section layout:
    • card-paragraph-options__header - Contains icon and heading
    • card-paragraph-options__content - Contains body text and button
  • Icon Display: Uses icon-display molecule with data-icon and data-color attributes
  • Icon Colors: Use data-color="navy" for default/gray backgrounds, data-color="white" for reverse
  • Available Icons: accelerate, attract, contact, delight, download, efficiency, engage, email, expertise, inbound, momentum, persona, roi, strategy, value, webinar
  • Icon Size: Large variant (90×90px) from icon-display molecule
  • Card Sizes:
    • DesktopPlus: 600px (Large)
    • Desktop: 500px (Medium)
    • Tablet: 400px (Small)
    • Mobile: 320px

JavaScript Icon Control

// Toggle icon visibility
function toggleIcon(show) {
    document.querySelectorAll('.card-paragraph-options')
        .forEach(card => card.setAttribute('data-show-icon', show));
}

// Change icon type (maintains color variant)
function changeIcon(iconName) {
    document.querySelectorAll('.icon-display')
        .forEach(icon => {
            const card = icon.closest('.card-paragraph-options');
            const isReverse = card.getAttribute('data-color') === 'reverse';
            const color = isReverse ? 'white' : 'navy';
            
            icon.setAttribute('data-icon', iconName);
            icon.setAttribute('data-color', color);
            icon.className = `icon-display icon-display--large icon--${iconName}-${color}`;
        });
}
