Usage Notes

Component Structure

  • 12 variants total: 4 sizes (Large, Medium, Small, Mobile) Ã— 3 colors (Default, Gray, Reverse)
  • Data attributes: Use data-size and data-color to control variants
  • Button component: References atomic button styles with hover and active states

Size Variants

  • Large: Max-width 600px, full text sizing
  • Medium: Max-width 500px, same text as Large
  • Small: Max-width 400px, same text as Large
  • Mobile: Max-width 380px, reduced heading size (21px), compact button padding

Color Schemes

  • Default: White background, navy text, navy button with white text
  • Gray: Light gray background (#eceeef), navy text, navy button with white text
  • Reverse: Navy background, white heading, muted body text (#c7cbcf), gold button with navy text

CSS Dependencies

  • Requires variables from variables-v2.0-20250915.css
  • Button states reference atomic button component styles
  • Typography uses Lato (headings/buttons) and Open Sans (body text)

Implementation Example

<div class="card-teaser" data-size="large" data-color="default">
    <div class="card-teaser__heading">
        <p>Your Heading Text</p>
    </div>
    <div class="card-teaser__body">
        <p>Your body content here...</p>
    </div>
    <a href="/landing-page" class="card-teaser__button">
        <div class="card-teaser__button-text">
            <p>Learn More</p>
        </div>
    </a>
</div>
