Usage Instructions

Note: This component uses the decorative-icon-display molecule for icon rendering. Ensure the sprite sheet CSS is loaded.

HTML Structure

<div class="pillar-card"> <div class="pillar-card__icon"> <div class="decorative-icon-display decorative-icon-display--xxlarge" data-icon="strategy" data-color="navy"></div> </div> <div class="pillar-card__content"> <div class="pillar-card__headline"> <h2 class="pillar-card__heading">Your Heading</h2> </div> <div class="pillar-card__paragraph"> <div class="pillar-card__text"> <p>Your content here</p> </div> </div> </div> </div>

Available Icon Options

  • Icons: accelerate, attract, contact, delight, download, efficiency, email, engage, exit, expertise, inbound, momentum, news, persona, professional, roi, schedule, strategy, unit, value, webinar
  • Colors: navy, gray, dark, white
  • Sizes: xxlarge (256px) for desktop, xlarge (120px) for mobile (automatic via responsive CSS)

Responsive Behavior

  • Mobile (380-767px): Vertical stack, icon above content, 16px gap
  • Tablet (768-1149px): Horizontal layout, icon left, content right, 32px gap
  • Desktop (1150-1919px): Horizontal layout, max-width 1264px for content
  • DesktopPlus (1920px+): Horizontal layout, max-width 1262px for content
